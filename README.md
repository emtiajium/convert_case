# What is this repository for?

A desktop-based app for the web-app <https://convertcase.net/>

# How to Run

###### Prerequisites

➜ Install Node 10 using [nvm](https://github.com/nvm-sh/nvm)

###### Clone the repo and install all dependencies

➜ `git clone git@github.com:emtiajium/convert_case.git`

➜ `cd convert_case`

➜ `npm install`

###### Start the development environment

➜ `npm run start`

###### Generate Windows package

➜ `npm run package-win`

###### Generate Linux package

➜ `npm run package-linux`
